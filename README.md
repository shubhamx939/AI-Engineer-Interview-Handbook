# AI Engineer Interview Handbook

A lightweight handbook and web reference for preparing AI engineer interviews, research rounds, and technical discussions.
Hosted at : https://shubhamx939.github.io/AI-Engineer-Interview-Handbook/

## About

This repository hosts a simple static site and reference material for AI engineering interview preparation. It is designed to help candidates review key topics, frameworks, and problem-solving strategies for roles that combine machine learning, systems design, and AI product engineering.

## Contents

- `index.html` — the main web page for the handbook.
- `README.md` — this overview and repository documentation.

## How to use

1. Open `index.html` in your browser.
2. Review the sections and navigation on the page.
3. Use this repository as a personal study guide or interview war room.

## Development

This is a static HTML project and does not require a build step.

To preview locally:

- Open `index.html` directly in a browser, or
- Use a simple static server such as `python3 -m http.server` from the repository root.
